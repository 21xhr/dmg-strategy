# IMAGO YouTube Transcript Corpus

**Snapshot date**: 2026-04-25

## Purpose

This folder holds plain-text transcripts imported from the local Downloads transcript folder.

The import target is the Evolucare YouTube channel and the selected videos are the ones whose titles or descriptions pointed clearly to IMAGO or closely related IMAGO product surfaces.

The original RTF files were converted to plain text so the corpus is easier to scan, quote, and compare against the IMAGO analysis note.

## Use

- scan for product language that repeats across videos
- look for claims that reinforce or weaken the current IMAGO vs DMG read
- extract vocabulary that may belong in the strategy glossary
- identify any recurring sector pain that differs from the Eric call notes
