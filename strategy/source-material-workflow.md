# Source Material Workflow

## Purpose

This document defines how to handle messy strategy and brainstorm material professionally.

## Recommended structure

Use three layers:

### 1. Raw inbox

One messy capture file is acceptable.

Its purpose is:

- fast idea capture
- unfiltered brainstorming
- temporary wording

It is not the source of truth.

### 2. Extracted strategy docs

As soon as a topic becomes durable, extract it into a focused tracked document.

Examples:

- market verticals
- internal organization use cases
- moat and portable identity
- gamification logic
- competitive positioning
- participation and adoption framing
- audience-specific pitch notes

These extracted docs are the material that should be readable, reusable, and eventually shareable.

### 3. Local-only notes and pointers

If you want to keep personal prompts, reminders, or raw external-source references, keep them local-only.

Good places:

- `.env` for simple local pointers or file references
- `.git/info/exclude` for local ignore rules that should not be pushed
- untracked working notes when they are still too raw for the repo

## Practical rule

Push cleaned strategy documents.

Do not push every raw brainstorm artifact by default.

Promote ideas from inbox to tracked docs only when they have been clarified enough to be useful later.

## Suggested cleanup of the current brainstorm doc

The raw brainstorm file should now be reduced rather than expanded.

Good cleanup moves:

- keep only unresolved prompts, wording tests, and half-formed ideas in the inbox
- cut any section that starts duplicating a tracked strategy or fundraising note
- delete duplicate headings and repeated blocks that were useful during ideation but no longer add information

What should stay in the raw file:

- unresolved wording options
- experimental examples not yet promoted into tracked docs
- half-formed ideas you still want to challenge later

What should leave the raw file:

- any section that is now duplicated in a clean strategy document with no important difference
